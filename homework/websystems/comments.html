Technical Aspects

There are lots of CSS Techniques used when I built this website. Three types of selectors are used: tag selector, class selector and id selector.
Tag Selectors which are which are body html ul li p h1 and a tags are used to Initialize the browser’s CSS stylesheet. There are two reasons for doing this. Firstly, the default value of some tags in different browsers is different because of browser compatibility issues.If we don’t do this, there will be differences between pages displayed in different browsers.Secondly, it can improve the quality of coding. If we don’t do this, there will be lots of duplicated CSS styles.
There are many class selectors which are sidebar aside-main article article-header title date article-entry table classes used to build the website’s pages. The sidebar class and article class are used to build the main structure of every page.We use these classes to avoid repeating defining css styles.The four pages have a same structure, and we can use css classes to build these page easily.
The id selector is main id, which represents the main section of the page.

Aesthetics

There are many aesthetics techniques used to build the pages.
  • 1. Match Colors Appropriately. The main color of the pages are black color , and use pink color to  emphasize important title and infos.
  • 2. Balance of page layout. I divide the page into two parts, the left and right part. The left part has a width of 300 px, and the right part extends to all full viewport. It has a good width factor.
